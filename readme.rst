GenericGraphPlugin
==================

Generic graph data struct plugin for ue4

Feature
-------

* Custom asset type
* UE4 BehaviorTree-like asset editor
* Extendable graph node type

Usage
-----

* SKill tree system
* Civilization-like technologies system
* etc
